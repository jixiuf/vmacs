* rime 白鹭版(egret) 输入方案
核心卖点: 五笔*拼音整句混输方案 (同一句话可可同时使用五笔与拼音)
| egret_wubi_py_sentence | 五笔*拼音整句混输        |
| egret_wubi_sentence    | 五笔整句输入(可拼音反查) |
| egret_wubi_py          | 五笔拼音                 |
| egret_wubi             | 五笔                     |
| egret                  | 全拼                     |
* 参考
 雾凇拼音 https://github.com/iDvel/rime-ice
